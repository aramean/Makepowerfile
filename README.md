# MakePowerFile
The most powerful makefiles ever made for you environments. Instant transform your project into modern DevOps.

Support:
- [x] Docker
- [x] Dependency management
- [x] CLI administration tools
- [x] CLI web server

## How to install?

```bash
$ git clone git@github.com:aramean/makepowerfile.git
```

### Go
```bash
$ make make:install go
```

### PHP

General:
```bash
$ make make:install php
```

WordPress:
```bash
$ make make:install php-wp
```

Drupal:
```bash
$ make make:install php-dp
```
